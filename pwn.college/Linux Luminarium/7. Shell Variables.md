## Printing Variables
```
export $FLAG
```
## Setting Variables
```
export PWN=COLLEGE
```
## Multi-word Variables
```
export PWN="COLLEGE YEAH"
```
## Exporting Variables
```
export PWN=COLLEGE && COLLEGE=PWN && /challenge/run
```
## Printing Exported Variables
```
env
```
## Storing Command Output
```
PWN=$(/challenge/run) && echo $PWN
```
## Reading Input
```
read PWN
```
## Reading Files
```
read PWN < /challenge/read_me
```
